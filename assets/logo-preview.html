ServalSheets Logo

To save as PNG:

  1. Right-click the logo above
  2. Select "Copy Image"
  3. Open Preview.app → File → New from Clipboard
  4. File → Export → Format: PNG → Save as "servalsheets-logo-120.png"

Or use this command:

brew install imagemagick && convert servalsheets-logo.svg -resize 120x120 servalsheets-logo-120.png
